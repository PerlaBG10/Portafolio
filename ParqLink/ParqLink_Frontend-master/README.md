# ParqLink Frontend Information:
    ParqLink Frontend is developed in the Android Studio application where users can search for nearby parking lots through an External Google Maps API, create a session in the parking lot through NFC, receive notifications, alerts in real time, etc.
# Techonlogies and Important Libraries used:
    Android Studio
    Google Maps API
    LottieFiles
    Retrofit
# Features:
    User Authentication (Login/Register): Secure login via email and password and JWT token storage for session management (provided by backend).
    
    Search and Display Nearby Parkings: Real-time map with available parkings based on user location.
    
    Details include name, address, price, schedule, and availability.
    
    Filtering and Sorting Options: Filter parkings by price, distance. Mark and access favorite parkings.
    
    Push Notifications and Alerts: Time-left alerts for active parking sessions.
    
    Parking Sessions via NFC: Start or end a session by scanning an NFC tag at the parking lot and real-time tracking of duration and cost.
    
    User Profile and History: View previously used parking locations and Manage personal preferences and favorites.
