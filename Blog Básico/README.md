# Proyecto Blog - Sitio Web Personal - Evaluación 2

## Descripción:

Este proyecto consiste en la creación de un sitio web con un menú de navegación horizontal y un submenú vertical.  
El objetivo principal es desarrollar el apartado **Blog**, mientras que el resto de secciones se muestran únicamente como enlaces en el menú.

El sitio incluye una barra de búsqueda, un área principal para publicaciones (posts) y una barra lateral con información adicional.

## Tecnologías utilizadas:

- **HTML5** → estructura del sitio web  
- **CSS3** → estilos y diseño visual  

## Estructura del proyecto:
- index.html <- página principal.
- css/estilos.css <- fichero con estilos para la página.
- README.md <- información sobre el proyecto.

## Características principales:

### Encabezado:
- Título del sitio web
- Barra de búsqueda

### Menú de navegación:
- Menú horizontal con enlaces
- El apartado **Blog** se muestra como activo mediante un `id`

### Sección principal (Blog):
- Contiene varios artículos (posts)
- Cada post incluye:
  - Título
  - Fecha y hora
  - Diseño
  - Nombre y Apellido
  - Añadir comentario
  - Texto descriptivo
  - Enlace Seguir leyendo…

### Barra lateral:
Incluye:
- Información personal (Sobre mí)
- Categorías
- Archivo
- Enlaces

### Pie de página:
- Muestra el nombre del autor y el año
